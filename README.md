# Examen_Parcial_Optimizaci-n-bio-inspirada-para-el-ajuste-de-hiperpar-metros-en-modelos-de-detecci-n
Uso de algoritmos metaheurísticos para maximizar la precisión de los clasificadores en entornos dinámicos.

La hiperconectividad de la Industria 4.0 ha posicionado a las redes IoT como infraestructuras críticas altamente vulnerables a ciberataques. Aunque los Sistemas de Detección de Intrusiones (IDS) basados en Machine Learning son efectivos, suelen presentar una alta Tasa de Falsos Positivos (FPR) al procesar datos desbalanceados, bloqueando tráfico legítimo y provocando interrupciones operativas (denegaciones de servicio autoinfligidas). Para resolver esta problemática, esta investigación presenta una arquitectura híbrida que integra el algoritmo predictivo Random Forest (RF) con la metaheurística de optimización Colonia de Abejas Artificiales (ABC). El objetivo principal es optimizar el ajuste de hiperparámetros aplicando un aprendizaje sensible al costo y una función de aptitud penalizada para configuraciones que superen un 20% de FPR. Los resultados empíricos, procesando un espacio reducido de 27 características, demostraron que el modelo propuesto (ABC-RF) logró disminuir el FPR del 43.11% (modelo estándar) al 23.04%, salvaguardando un volumen significativo de paquetes operativos críticos sin comprometer la detección de amenazas. Además, el optimizador convergió hacia una estructura computacionalmente ligera (20 estimadores, profundidad máxima de 16), resultando idónea para despliegues de baja latencia en dispositivos de borde (Edge Computing). Se concluye que la optimización bio-inspirada resuelve el trade-off histórico entre seguridad y disponibilidad, garantizando la continuidad del negocio en los entornos industriales de 2026.
